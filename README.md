# lua-language-server

[![Build status](https://ci.appveyor.com/api/projects/status/0tng1g72fssvu9rr/branch/master?svg=true)](https://ci.appveyor.com/project/sumneko/vscode-lua-language-server/branch/master)

## Feature

- [x] Goto Definition
- [x] Goto Implementation
- [x] Find All References
- [x] Type Inference
- [x] Hover
- [x] Diagnostics
- [x] Rename
- [x] Auto Completion
- [x] IntelliSense
- [x] Signature Help
- [x] Document Symbols
- [x] Support Dirty Script
- [x] Syntax Check
- [x] Search Globals
- [x] Highlight
- [x] Code Action
- [ ] Multi Workspace
- [ ] Type Format

## Locale

- [x] en-US
- [x] zh-CN
